# Graphical Perception Study — Reviewer Demo

## Note on This Deployment

This deployment is prepared for manuscript reviewers. It presents each graphical perception experiment once in a fixed order so the study flow, instructions, stimuli, and response controls can be inspected without running the full participant-length study.

Please proceed through the tasks as a participant would: read each section introduction, enter a response in the sidebar, and use **Next** or **Enter** to continue. You can also use the **study browser** on the right sidebar to see the full study structure and jump to any task directly.

---

## Study Structure

You will go through all four task types in the following order:

1. **Accuracy** — Estimate the value of a single visual channel (e.g., position, length, tilt, area, luminance, saturation, curvature). Enter a numerical value.
2. **Discriminability** — Judge whether two elements are the *same* or *different* along a given visual channel.
3. **Separability** — Same as Accuracy, but with an irrelevant visual channel varying simultaneously. This tests whether unrelated channels interfere with judgments.
4. **Pop-out** — An array of identical elements is briefly displayed; judge whether one outlier element is present.

Each section begins with an instruction page explaining the task.

---

## Demonstration-Specific Changes

To allow you to experience the full experimental pipeline efficiently, the following modifications have been made compared to the main study:

- **Trial count**: Only one trial per channel is included in each task. The main study used 1–32 trials per channel depending on the task.
- **Pop-out task**: The number of display elements has been reduced from 20 to 10, and the stimulus exposure time has been extended from 100 ms to 5 seconds, so that you can comfortably observe the stimulus without time pressure.
- **Consent form**: The consent form is omitted from this reviewer deployment because it is a non-participant demonstration. This omission is for review convenience only.
These changes are solely for demonstration purposes and **do not reflect the parameters used in our reported analyses**. The full experimental design and sample sizes are described in the paper.

---

## Requirements

- Normal color vision (no colorblindness)
- Familiarity with basic concepts: decimal numbers, fractions, angles, and areas
